### **Project PRD/Brief: Hybridcasual Puzzle Gameplay Interface - Sand Theme**

**Project Title:** Sand Qix

---

### **1. Executive Summary**

This document outlines the requirements for redesigning the core gameplay interface of our hybridcasual puzzle game. The primary goal is to enhance the visual appeal and "juiciness" of the experience by implementing a modern, high-feedback UI aesthetic. The previous iteration focused on "pink frosting" and "bubbles," but the direction has been revised to incorporate a vibrant and engaging "sand" theme while retaining key visual elements like modern glassmorphism and high-energy feedback.

### **2. Project Goal**

To create a highly engaging, visually "juicy" hybridcasual gameplay interface with a distinct "sand" theme, ensuring every player interaction feels rewarding and polished. This redesign aims to improve player satisfaction, visual clarity, and alignment with a potential broader sand/desert/beach game theme.

### **3. Target Audience**

Mobile gamers who enjoy hybridcasual puzzle games, appreciate vibrant visuals, clear feedback, and a polished user experience.

### **4. Scope**

This PRD focuses exclusively on the visual design and interactive elements of the in-game puzzle gameplay interface. It includes:
*   Puzzle element textures and visual feedback.
*   General UI style (buttons, progress bars, score displays).
*   Background environment.
*   Mascot integration.
*   Combo and feedback animations.

**Inclusions:**
*   Modern Glassmorphism UI style.
*   High visual feedback ("juice") for interactions.
*   Sand-themed puzzle elements and background.
*   Specific treatment for combo pop-ups and slice edges.
*   Mascot visual integration.

**Exclusions:**
*   Changes to core game mechanics or logic.
*   Sound design or haptic feedback.
*   Other game screens (e.g., main menu, shop, level complete - these would be separate follow-up projects).
*   Technical implementation details beyond visual specification.

### **5. Key Features & Requirements (Visual Design Focus)**

The interface will combine a high-energy, "juicy" feel with a cohesive sand theme, building upon the previously preferred "V3" style.

**5.1. Overall Aesthetic:**
*   **Hybridcasual "Juice":** Prioritize vibrant colors, polished UI elements, and high visual feedback to make every interaction feel rewarding.
*   **Modern Glassmorphism UI:** UI elements (e.g., score displays, progress bars) should feature a modern "glassmorphism" style with frosted transparency and subtle depth.
*   **Sand Theme Integration:** All visual elements must be re-imagined to connect with a sand, desert, or beach environment.

**5.2. Puzzle Elements (Sliced Sections):**
*   **Texture:** Replace "pink frosting" with a gritty, sparkling sand texture. (Specific variant details below).
*   **Glow:** Maintain "white-hot glowing trails/edges" for sliced sections to provide clear and satisfying visual feedback, enhancing the "juicy" feel.
*   **Interaction Feedback:** Consider subtle particle effects (e.g., sand grains flying) upon slicing.

**5.3. UI Elements (Top Bar, Pop-ups):**
*   **Standard UI:** Top UI elements (lives, progress bar, score) should adopt the modern glassmorphism style.
*   **Combo Pop-ups:** "Combo" text pop-ups should be retained, with animated, high-energy visual flair. (Specific variant details below).

**5.4. Background:**
*   The background should be a cohesive sand-themed environment. (Specific variant details below).

**6.1. Golden Sand Puzzle Gameplay V1:**
*   **Puzzle Piece Texture:** Gritty, sparkling golden sand.
*   **Background:** Warm, desert-dusk gradient with subtle sand dune silhouettes.
*   **Combo Text:** Sun-kissed orange and yellow gradient style for pop-ups.

### **7. Success Metrics**

*   **Design Alignment:** Visual designs successfully embody the "juicy" hybridcasual feel, modern glassmorphism, and the chosen sand theme.
*   **Positive Internal Feedback:** Design team, product team, and stakeholders agree on the visual appeal and effectiveness of the new interface.
*   **Clarity & Feedback:** Interactions are clear, and visual feedback is immediate and satisfying.
